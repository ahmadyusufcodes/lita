Open Source Monitor Light

Turn Any Monitor Into a Clean, Controlled Studio Light

Lita Light is a desktop white screen and color-temperature light app for creators, streamers, and remote teams. It delivers true flat full-screen light with fast controls and no clutter.

Download Artifacts View Source

Keywords: monitor light app, white screen app, desktop softbox, fullscreen fill light.

What You Get

True Full-Screen Light

Flat color output without texture overlays, optimized for practical on-camera fill light use.

Temperature + Brightness

Tune from warm to cool (`1800K-9000K`) and adjust brightness quickly in live scenarios.

Fast Interaction Model

Scroll to change brightness, Shift + Scroll to change temperature, and drag the controls panel anywhere.

Always on Top

Keep your light over other windows during calls, recording, and content sessions.

Supported Platforms

  • Linux x64
  • Windows x64
  • macOS Intel
  • macOS Apple Silicon

Every push runs a GitHub Actions pipeline that builds platform bundles and publishes downloadable artifacts.

Built with Tauri, TypeScript, and Rust by Ahmad Yusuf.
